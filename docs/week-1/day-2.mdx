# 12-Feb-2025
## பிலிப்பியர் 1:27
`எவ்விதத்திலும் நீங்கள் கிறிஸ்துவின் சுவிசேஷத்திற்குப் பாத்திரராகமாத்திரம் நடந்துகொள்ளுங்கள்.`

|          |         |
| -------- | ------- |
| எவ்விதத்திலும் | நன்மையிலும், தீமையிலும், உயர்விலும், தாழ்விலும், மகிழ்ச்சியிலும், துன்பத்திலும் |
| நீங்கள் | நான், என்னை சார்ந்தவர்கள்  |
| கிறிஸ்துவின் | இயேசு கிறிஸ்து |
| சுவிசேஷத்திற்குப் | நற்செய்தி, புகழ்ச்சி |
| பாத்திரராகமாத்திரம் | என்னை கொண்டு, என் மூலமாக |
| நடந்துகொள்ளுங்கள் | என்னுடைய செய்கைகளை கொண்டு எப்பொழுதும் |

## செயல்பாடு
* சுவிசேஷத்திற்கு அனுதின வாழ்வில் என்னுடைய தனிப்பட்ட பங்களிப்பு அளிக்க 
* எண்ணில் சுவிசேசத்திக்கு தடையாய் இருக்கிற எல்லா செயல்கள்

## ஜெபம்
இயேசுவே நான் எல்லா விதத்திலும் சுவிசேஷத்திற்கு நல்ல பாத்திரமாக கருவியாக செயல்பட அர்ப்பணிக்கிறேன் 
எல்லா சூழ்நிலையிலும் உம்முடைய நற்செய்தியை பிறருக்கு சொல்ல நோக்கமாக செயல்படனும், சுவிசேஷத்திற்கு எவ்விதத்திலும் தடையோ தாமதமோ என் மூலம் வராமல் எல்லா விதத்திலும்
 என்னை உபயோகமாக பயன்படுத்தும்
