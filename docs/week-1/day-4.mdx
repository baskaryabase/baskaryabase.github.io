# 14-Feb-2025
## பிலிப்பியர் 3:12
`கிறிஸ்து இயேசுவினால் நான் எதற்காகப் பிடிக்கப்பட்டேனோ அதை நான் பிடித்துக்கொள்ளும்படி ஆசையாய்த் தொடருகிறேன்.`

|          |         |
| -------- | ------- |
| கிறிஸ்து | புது சிருஷ்டி, புதிய மனிதன், பரிசுத்தர் |
| இயேசுவினால் | இயேசு கிறிஸ்து |
| நான் | என்னை |
| எதற்காகப் | தேவ சித்தம், ரட்சிப்பு, நித்திய ஜீவன் |
| பிடிக்கப்பட்டேனோ | பிடிக்கப்பட்டேனோ |
| அதை | ஆசை, பிற காரியத்தின் மீது வெறுப்பு |
| நான் | என்னை, என் குடும்பம், என் நண்பர்கள் |
| பிடித்துக்கொள்ளும்படி | என்னுடைய செயல்பாடுகள், குணாதிசியங்கள், சிந்தனை, பார்வை |
| ஆசையாய்த் | விருப்பத்தோடு  |
| தொடருகிறேன் | ஒழுங்காய், ஒரே சிந்தையோடு, தொடரனும்  |

அதிகம் தோட்ட வார்த்தை -> தொடருகிறேன் 

## செயல்பாடு
* தினமும் ஜெபத்தில் தேவனோடு உள்ள உறவில் பலப்படணும்
* ஒவ்வொரு நாளும் அவர் என் மீது வைத்த சித்தத்தை அறிந்துகொள்ள
* அதிகம் வசனத்தை தியானிக்கனும்

## ஜெபம்
நீர் என் மீது வைத்துள்ள திட்டத்தின் படி தினம் தினம் ஓட, பாவத்திற்கு விலக, பரத்தை நாட, பரிசுத்தமாய் மாற, உலகை வெறுக்க உதவி செய்யும் அர்ப்பணிக்கிறேன் என்னை வனைந்து உருவாக்குமே
