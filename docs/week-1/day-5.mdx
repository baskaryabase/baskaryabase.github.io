## பிலிப்பியர் 4:6
`நீங்கள் ஒன்றுக்குங்கவலைப்படாமல், எல்லாவற்றையுங்குறித்து உங்கள் விண்ணப்பங்களை ஸ்தோத்திரத்தோடே கூடிய ஜெபத்தினாலும் வேண்டுதலினாலும் தேவனுக்குத் தெரியப்படுத்துங்கள்.`

|          |         |
| -------- | ------- |
| நீங்கள் | நான் |
| ஒன்றுக்குங்கவலைப்படாமல் | உணவு, உடை, உடல் சுகம், தேவை, குறைவு, மனமடிவு, பிறர் சொல்  |
| எல்லாவற்றையுங்குறித்து | கர்த்தருக்காய் பயன்பட, பரிசுத்தமாய் வாழ, நித்தியத்தில் சேர |
| உங்கள் | என் சூழ்நிலை, எதிர்பார்ப்பு, முயற்சி |
| விண்ணப்பங்களை | சொல்லுவது, வேண்டுகோள் |
| ஸ்தோத்திரத்தோடே | துதி, நன்றி செலுத்தல் |
| கூடிய | நல்மனதோடு, செயல்பாடோடு |
| ஜெபத்தினாலும் | ஜெபம் |
| வேண்டுதலினாலும் | சொல்லுவது புகார்  |
| தேவனுக்குத் | இயேசுவுக்கு |
| தெரியப்படுத்துங்கள் | சொல்லவேண்டும் |

அதிகம் தோட்ட வார்த்தை -> ஒன்றுக்குங்கவலைப்படாமல் எல்லாவற்றையுங்குறித்து

## செயல்பாடு
* அனுதின வாழ்வில் சிறு சிறு காரியத்திற்காக கவலை படுவதை தவிர்க்க
* நாம் பிறரிடம் பேசுவது, மற்றும் பிறர் நம்மிடம் பேசுவது ஆகியவற்றில் தேவனுக்கு தெரியப்படுத்தல்

## ஜெபம்
அநேக நேரம் என் தேவை, என் குறைவு, நான் எண்பதயே அதிகம் கவலை படுத்துகிகிறது என் தேவை கவலையை மறந்து உம் தேவையை கவலையை உணர்ந்து அதற்கு நான் பயன்படும் பாத்திரமாக நடக்கவேண்டும்.
