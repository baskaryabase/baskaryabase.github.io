# 16-Feb-2025
## பிலிப்பியர் 4:13
`என்னைப் பெலப்படுத்துகிற கிறிஸ்துவினாலே எல்லாவற்றையுஞ்செய்ய எனக்குப் பெலனுண்டு.`

|          |         |
| -------- | ------- |
| என்னைப் | எனக்கு |
| பெலப்படுத்துகிற | மனஉறுதி, உற்சாகம், ஆரோக்கியம், சாமர்த்தியம், பெலன் |
| கிறிஸ்துவினாலே | தனிமையில் என்னோடு இருப்பவர், எனக்காக யுத்தம் செய்பவர் |
| எல்லாவற்றையுஞ் | என் வாழ்வின் செயல் |
| செய்ய | செயல்பாடு |
| எனக்குப் | எனக்குள் இருப்பவர் எனக்கு பெலன் தருபவர் |
| பெலனுண்டு | காரியத்தை செய்ய பெலன் |

அதிகம் தோட்ட வார்த்தை -> பெலப்படுத்துகிற கிறிஸ்துவினாலே

## செயல்பாடு
* குடும்பத்தில் ஒரு முக்கிய முடிவு எடுக்கும் போது சில நேரங்களில் செய்ய யோசிப்பதுண்டு வாழ்க்கை பிழைப்புக்கு ஒரு முடிவு எடுத்து செயல் பட
* ஒரு சில ஒழுங்குக்கு முயற்சி செய்வது, ஒரே சிந்தையாயிருப்பது அதில் நிலைத்திருப்பது, போன்ற காரியங்களில் கிறிஸ்துவின் பெலனை சார்ந்து கொள்ள விரும்புகிறான்

## ஜெபம்
உம்மில் விசுவாசமாய் இருக்க, என் செயல்களில் உம்முடைய பெலன், கற்று கொண்டவைகளில் நிலைத்திருக்க, ஒரே ஒழுங்காய் ஓட, ஒரே சிந்தையாயிருக்க பெலன் தங்கப்பா

2. அநேக நேரங்களில் பெரிய காரியங்களில் தான் தேவனிடம் சொல்லி ஜெபித்து தீர்மானிக்கிறான் சில நேரங்களில் சிறு சிறு காரியத்திற்கு நானாக முடிவெடுப்பதுண்டு தேவனிடம் சொல்லாமல் அதை சரி செய்து கொள்ள கற்றுக்கொள்கிரேன்
