## சங்கீதம் 1:1
`கர்த்தருடைய வேதத்தில் பிரியமாயிருந்து, இரவும் பகலும் அவருடைய வேதத்தில் தியானமாயிருக்கிற மனுஷன் பாக்கியவான்.`

|                  |                                                                         |
| ---------------- | ----------------------------------------------------------------------- |
| கர்த்தருடைய      | தேவனுடைய                                                                |
| வேதத்தில்        | கர்த்தருடைய இருதயம், இயேசுவுடைய வாழ்க்கை, பரிசுத்த ஆவியின் செயல்பாடுகள் |
| பிரியமாயிருந்து  | அதை ஆசையை நாடி தேடுகிற இருதயம், மற்ற காரியங்களை வெறுத்தல்               |
| இரவும்           | படுக்கைக்கு செல்லும் முன், தூங்கும் போது கனவில்                         |
| பகலும்           | எழுந்தவுடன், நாள் முழுதும்                                              |
| அவருடைய          | கர்த்தருடைய                                                             |
| வேதத்தில்        | கர்த்தருடைய இருதயம், இயேசுவுடைய வாழ்க்கை, பரிசுத்த ஆவியின் செயல்பாடுகள் |
| தியானமாயிருக்கிற | நினைவாகவே, கவனமாக, நம் வாழ்க்கையோடு வேதத்தை ஒப்புமை படுத்தி             |
| மனுஷன்           | நான், என்னை                                                             |
| பாக்கியவான்      | ஆசீர்வதிக்க பட்டவன்                                                     |

அதிகம் தோட்ட வார்த்தை -> தியானமாயிருக்கிற 

## செயல்பாடு
- வாசித்த வசனத்தை மறக்காமல் நாள் முழுவதும் வசனத்தை வாழ்க்கையில் நடக்கும் சம்பவங்களோடு ஒப்பிட்டுப் பார்த்து எப்பொழுதும் வசனத்தின் நினைவாகவே இருக்க வேண்டும் என்று உணர்த்தப்படுகிறேன்

## ஜெபம்
இதோ எனக்குள் வாசமாய் இருக்கிற பரிசுத்த ஆவியானவரே ஒவ்வொரு நாளும் உம்முடைய வசனத்தில் தியானமாய் இருக்க கிருபை தாரும் சிந்தையாய் மட்டுமல்ல அதை வாழ்க்கையில் செயல்படுத்தவும் உதவி செய்யும்
