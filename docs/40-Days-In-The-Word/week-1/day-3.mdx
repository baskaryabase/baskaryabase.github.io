# 13-Feb-2025
## பிலிப்பியர் 2:13
`ஏனெனில் தேவனே தம்முடைய தயவுள்ள சித்தத்தின்படி விருப்பத்தையும் செய்கையையும் உங்களில் உண்டாக்குகிறவராயிருக்கிறார்.`

|          |         |
| -------- | ------- |
| ஏனெனில் | இரட்சிப்பு நிறைவேற, சுவிசேஷம் அறிவிக்கப்பட, அன்றாட பிழைப்புக்கு |
| தேவனே | இயேசு கிறிஸ்துவே |
| தம்முடைய | தேவ திட்டம் |
| தயவுள்ள | நல்ல காரியம், நம்மால் இயன்ற, சாத்தியபடும் காரியம் |
| சித்தத்தின்படி | திட்டப்படி |
| விருப்பத்தையும் | ஆசை, பிற காரியத்தின் மீது வெறுப்பு |
| செய்கையையும் | என்னுடைய செய்கைகளை கொண்டு எப்பொழுதும் |
| உங்களில் | நம் வாழ்வில், குணங்களில், பேச்சில், செயல் |
| உண்டாக்குகிறவராயிருக்கிறார் | அனுமதிக்கிற காரியம், கற்றுக்கொடுக்கிறார், சூழ்நிலைகளை ஏற்படுத்துகிறார், புரிதலையும் உணர்த்துதலையும் நமக்கு தருகிறார், அதற்கு மாறாக இருக்கும் என் செயல்களை தவிர்க்க  |

## செயல்பாடு
* என்னுடைய அனுதின வாழ்வின் நோக்கம் செயல்பாடுகள் எல்லாவற்றிலும் தேவன் எனக்கு கற்றுத்தருகிற என் வாழ்வில் அனுமதிக்கிற ஏற்படுத்துகிற சூழ்நிலைகள் வாய்ப்புகள் எல்லாவற்றிலும் முழு மனதோடு நான் என் பங்களிப்பை கொடுப்பேன்

## ஜெபம்
இயேசுவே நீர் எனக்கு கொடுக்கும் எல்லா வாய்ப்புகளையும் சரிவர நான் பயன்படுத்த அலட்சியமில்லாமல் செயல்பட என் வாழ்வில் உம் சித்தம் செய்ய என்னை அர்ப்பணிக்கிறேன்.
